ENGLISH:
---ENIGMA MACHINE---
-INSTRUCTION MANUAL-

Encrypt:
Step 1- Enter the encryption key, there are 3 ways to do this:
a) Manually configure the settings
b) Get a randomly generated encryption key
c) Directly enter the encryption key

Step 2- Enter text to encrypt
Step 3- Get the encrypted text

Decrypt:
Step 1- Directly enter the key used for encryption
Step 2- Enter the encrypted text
Step 3- Get the decrypted original text

Key:
-First 3 numbers in the key indicate the rotors used and their order
-The original enigma machine had 5 rotors, so the first 3 numbers in
 the key can be any digit between 1 and 5.

-Next 6 digits in the key indicate the position of each of these rotors
-The original enigma machine had 26 positions for each rotor, due to the
 26 letters of the alphabets. Hence these digits can be any value between
 00 and 25

-The last 20 digits in the key indicate the plugs connected, these are the
 connections between alphabets for further encryption possibilities.
-These digits can have any values between A and Z (case insensitive) 
 and the number of plugs used must be a multiple of 2.


DEUTSCH:
---ENIGMA-MASCHINE---
-BEDIENUNGSANLEITUNG-

Verschlüsseln:
Schritt 1 – Geben Sie den Verschlüsselungsschlüssel ein. Es gibt drei Möglichkeiten, dies zu tun:
a) Konfigurieren Sie die Einstellungen manuell
b) Holen Sie sich einen zufällig generierten Verschlüsselungsschlüssel
c) Geben Sie den Verschlüsselungsschlüssel direkt ein

Schritt 2: Geben Sie den zu verschlüsselnden Text ein
Schritt 3 – Holen Sie sich den verschlüsselten Text

Entschlüsseln:
Schritt 1: Geben Sie den für die Verschlüsselung verwendeten Schlüssel direkt ein
Schritt 2: Geben Sie den verschlüsselten Text ein
Schritt 3 – Holen Sie sich den entschlüsselten Originaltext

Schlüssel:
-Die ersten 3 Zahlen im Schlüssel geben die verwendeten Rotoren und deren Reihenfolge an
-Die ursprüngliche Enigma-Maschine hatte 5 Rotoren, also die ersten 3 Zahlen
 Der Schlüssel kann eine beliebige Ziffer zwischen 1 und 5 sein.

-Die nächsten 6 Ziffern im Schlüssel geben die Position jedes dieser Rotoren an
-Die ursprüngliche Enigma-Maschine hatte aus diesem Grund 26 Positionen für jeden Rotor
 26 Buchstaben des Alphabets. Daher können diese Ziffern einen beliebigen Wert dazwischen haben
 00 und 25

-Die letzten 20 Ziffern im Schlüssel geben die angeschlossenen Stecker an, das sind die
 Verbindungen zwischen Alphabeten für weitere Verschlüsselungsmöglichkeiten.
-Diese Ziffern können beliebige Werte zwischen A und Z haben (Groß- und Kleinschreibung wird nicht beachtet)
 und die Anzahl der verwendeten Stecker muss ein Vielfaches von 2 sein.
